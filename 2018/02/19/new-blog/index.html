先弄个博客再说——记录浪费的生命。。。

Veröffentlicht am 2018-02-19 |

参考简书上代码咖啡的【20分钟教你使用hexo搭建github博客】对就是要这种短平快的。。

  1. 准备工具：macbook pro, github账号, Git
  2. create repository : mylvcs.github.io
  3. 生成秘钥保留在本地： ssh-keygen -t rsa -C “github邮箱地址”
  4. 生成的秘钥.pub那个复制黏贴到刚created的depository 的ssh地方。
  5. 本地安装node.js (这里本人遇到的坑是需要sudo npm install …)
  6. 下载安装hexo
    终端里敲：$npm install -g hexo-cli
  7. 初始化hexo 找个地方放你的博客本地目录
    $hexo init
    $cd
    $npm install
  8. 里是配置文件，打开里面的_config.yml
  9. 改language, timezone,url
  10. 最重要是改最后的deploy
    type: git
    repo: https://github.com/mylvcs/mylvcs.github.io.git
    branch: master

Done.下面是发博客进行部署

  1. 终端hexo new “博客题目” 可以看到本地目录里source -> _post文件夹下有新建的md
  2. 修改并保存好本地的md后 $hexo server 进行本地发布
  3. 浏览器尝试打开成功后 $hexo generate
  4. $hexo deploy 这里遇到一个坑：deployer not found:git
    解决办法：$npm install hexo-deployer-git –save 再执行hexo deploy 成功！
  5. 绑定域名 打开域名提供商：我的是腾讯云
    控制台找到解析，添加解析：类型：CNAME 主机记录@， 记录值是域名：shuati.club
  6. 浏览器输入域名，出现404
  7. github depository 里新建一个文件名为CNAME, 内容是域名。
  8. 本地source文件夹下，新建名为CNAME文件，里面内容是域名：shuati.club
  9. 重新deploy后浏览器输入绑定域名 成功！
